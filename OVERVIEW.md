## 🐛 Troubleshooting

### Issue: Storage full
- Delete some photos
- Use compressed images
- Try smaller file sizes

### Issue: Photos not loading
- Check localStorage is enabled
- Try private browsing (may disable storage)
- Clear cache and reload

### Issue: Music not playing
- Click music button first (browser requires interaction)
- Check browser permissions
- Verify audio file URL is accessible

### Issue: Password not working
- Password is: `100324` (six digits)
- Check Caps Lock
- Try different browser

For more help, see **README.md** troubleshooting section.
