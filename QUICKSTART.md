# 🚀 Quick Start Guide

Get started with "Our Memories" photo album in 2 minutes!

## Opening the Website

### Windows / Mac / Linux (Desktop)

1. Open the `d:\Love\` folder
2. Right-click on `index.html`
3. Select "Open with" → Choose your browser (Chrome, Firefox, Safari, etc.)
4. Or simply double-click `index.html`

**That's it!** The website will open in your browser.

### Mobile Devices

**Android:**
1. Copy all files to your Android phone
2. Use a file manager or send via email
3. Open `index.html` with a browser
4. Or upload to a web server (free options: GitHub Pages, Netlify)

**iPhone:**
1. Find a simple way to access the files (email, cloud, etc.)
2. Open `index.html` in Safari
3. Or upload to a web server

### Step 1: Access the App

## Essential Features

### Upload Photos
- **Click:** "➕ Add Photos" button
- **Or:** Drag & drop photos anywhere on the page
- **Supported:** JPG, PNG, GIF, WebP
- **Limit:** ~10-20 photos (before storage gets full)

### View Photos
- **Click:** Any photo in gallery
- **Full-screen:** Lightbox mode opens
- **Navigate:** Arrow keys or buttons
- **Close:** Press Escape or click X

### Edit Photo Info
1. Click a photo to open lightbox
2. Click "✏️ Edit" button
3. Change date and add notes
4. Click "Save"

### Delete Photos
- Hover over photo
- Click "🗑️ Delete" button
- Confirm deletion

### Background Music
- Click "🎵 Music: Off" button
- Music will play automatically
- Click again to toggle off

## Important Tips

### 1. **Storage Location**
- Photos are saved in your browser's storage
- They stay on your device only
- Clearing browser history may delete photos!

### 2. **Backup Your Photos**
- Take screenshots of lightbox view
- Or download individual photos before browser update

### 3. **Browser Compatibility**
- Works best on: Chrome, Firefox, Safari, Edge
- Not supported: Internet Explorer

### 4. **Mobile Installation**

**Install as App (Android):**
1. Open website in Chrome
2. Tap menu (three dots)
3. Select "Install app"
4. Play Store opens
5. Click "Install"

**Install as App (iPhone):**
1. Open website in Safari
2. Tap Share button
3. Tap "Add to Home Screen"
4. Give it a name
5. Tap "Add"

## Troubleshooting

### Problem: "Can't see photos after I closed browser"

**Solution:**
- LocalStorage should persist data
- Check if you're using Private/Incognito mode (disables storage)
- Try a different browser

### Problem: "Upload not working"

**Solution:**
- Check file size (should be less than 5MB each)
- Try a different file format
- Clear browser cache and try again

### Problem: "Music won't play"

**Solution:**
- Click music button first (browsers require user interaction)
- Check if browser allows audio
- Try a different browser

### Problem: "Password incorrect"

**Solution:**
- Password is: `100324` (six digits)
- Make sure Caps Lock isn't on
- Try again

### Problem: "Website is slow"

**Solution:**
- Reduce image size before uploading
- Don't store 100+ photos
- Use compressed images (tools: tinypng.com)
